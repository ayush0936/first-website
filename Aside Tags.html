Understanding html lists

HTML lists are used to group a set of related items in a specific order. There are two main types of lists in HTML: ordered lists and unordered lists.

Tip from Ayush

HTML lists are used to group related items. You should remember to use the appropriate tags for the type of list you want to create. Use <ol> for ordered lists and <ul> for unordered lists.
